
## Выполненные задачи
1. Установила библиотеку jack-theripper/yandex, используя composer.<br>
2. Получила API токен для работы с Яндекс Диском и изучила соответственную документацию.<br>
3. Написала комплексный компонент `only:yandex.disk` и простые компоненты `only:yandex.disk.dir` и `only:yandex.disk.file` для реализации возможностей добавления, просмотра и удаления ресурсов. Для файлов текстового формата также реализована возможность изменения их содержимого. Примеры отображения (1) папки, (2) файла с изображением и (3) файла с текстом.<br>
